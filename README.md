# Flexci-Blackbox-Intelligence
Innovation Project: Edge-AI &amp; Satellite Tracking for Car Sharing

Flexci is a security system for car-sharing that combines Edge-AI for driver behavior analysis and Hybrid Connectivity (Cellular + Satellite) to ensure vehicles are always trackable.



✨ Key Features

- Hybrid Link: Automatically switches to Satellite when cellular signal is lost.

- Edge-AI Scoring: Calculates driver scores locally to reduce server load.

- Anti-Theft: Remote vehicle locking and unauthorized access detection.
